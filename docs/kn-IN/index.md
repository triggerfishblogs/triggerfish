## ಒಂದು ಆಜ್ಞೆಯಲ್ಲಿ ಸ್ಥಾಪಿಸಿ

::: code-group

```bash [macOS / Linux]
curl -sSL https://raw.githubusercontent.com/greghavens/triggerfish/master/scripts/install.sh | bash
```

```powershell [Windows]
irm https://raw.githubusercontent.com/greghavens/triggerfish/master/scripts/install.ps1 | iex
```

```bash [Docker]
docker run -v ./triggerfish.yaml:/data/triggerfish.yaml \
  -p 18789:18789 -p 18790:18790 \
  ghcr.io/greghavens/triggerfish:latest
```

:::

ಬೈನರಿ ಇನ್‌ಸ್ಟಾಲರ್‌ಗಳು ಪೂರ್ವನಿರ್ಮಿತ ರಿಲೀಸ್ ಡೌನ್‌ಲೋಡ್ ಮಾಡುತ್ತವೆ, ಅದರ ಚೆಕ್‌ಸಮ್ ಪರಿಶೀಲಿಸುತ್ತವೆ ಮತ್ತು
ಸೆಟಪ್ ವಿಝಾರ್ಡ್ ಚಲಾಯಿಸುತ್ತವೆ. Docker ಸೆಟಪ್, ಮೂಲದಿಂದ ನಿರ್ಮಿಸುವುದು ಮತ್ತು ರಿಲೀಸ್ ಪ್ರಕ್ರಿಯೆಗಾಗಿ
[ಸ್ಥಾಪನಾ ಮಾರ್ಗದರ್ಶಿ](/kn-IN/guide/installation) ನೋಡಿ.

API ಕೀಗಳನ್ನು ನಿರ್ವಹಿಸಲು ಬಯಸುವುದಿಲ್ಲವೇ? Triggerfish Gateway ಗಾಗಿ [ಬೆಲೆ ನೋಡಿ](/kn-IN/pricing) —
ನಿರ್ವಹಿಸಲಾದ LLM ಮತ್ತು ಸರ್ಚ್ ಮೂಲ ಸೌಕರ್ಯ, ನಿಮಿಷಗಳಲ್ಲಿ ಸಿದ್ಧ.

## ಇದು ಹೇಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ

Triggerfish ನಿಮ್ಮ AI ಏಜೆಂಟ್ ಮತ್ತು ಅದು ಸ್ಪರ್ಶಿಸುವ ಎಲ್ಲದರ ನಡುವೆ ನಿರ್ಧಾರಾತ್ಮಕ
ನೀತಿ ಪದರವನ್ನು ಇಡುತ್ತದೆ. LLM ಕ್ರಿಯೆಗಳನ್ನು ಪ್ರಸ್ತಾಪಿಸುತ್ತದೆ — ಶುದ್ಧ-ಕೋಡ್ hook ಗಳು ಅವುಗಳನ್ನು
ಅನುಮತಿಸಲಾಗಿದೆಯೇ ಎಂದು ನಿರ್ಧರಿಸುತ್ತವೆ.

- **ನಿರ್ಧಾರಾತ್ಮಕ ನೀತಿ** — ಭದ್ರತಾ ನಿರ್ಧಾರಗಳು ಶುದ್ಧ ಕೋಡ್. ಯಾವುದೇ ಯಾದೃಚ್ಛಿಕತೆಯಿಲ್ಲ,
  LLM ಪ್ರಭಾವವಿಲ್ಲ, ಯಾವುದೇ ಅಪವಾದವಿಲ್ಲ. ಒಂದೇ ಇನ್‌ಪುಟ್, ಒಂದೇ ನಿರ್ಧಾರ, ಪ್ರತಿ ಬಾರಿ.
- **ಮಾಹಿತಿ ಹರಿವು ನಿಯಂತ್ರಣ** — ನಾಲ್ಕು ವರ್ಗೀಕರಣ ಮಟ್ಟಗಳು (PUBLIC, INTERNAL,
  CONFIDENTIAL, RESTRICTED) session taint ಮೂಲಕ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪ್ರಸರಿಸುತ್ತವೆ. ಡೇಟಾ
  ಎಂದಿಗೂ ಕಡಿಮೆ ಸುರಕ್ಷಿತ ಸಂದರ್ಭಕ್ಕೆ ಕೆಳಗೆ ಹರಿಯಲು ಸಾಧ್ಯವಿಲ್ಲ.
- **ಆರು ಜಾರಿ Hooks** — ಡೇಟಾ ಪೈಪ್‌ಲೈನ್‌ನ ಪ್ರತಿ ಹಂತವನ್ನು ನಿಯಂತ್ರಿಸಲಾಗಿದೆ: LLM
  ಸಂದರ್ಭಕ್ಕೆ ಏನು ಪ್ರವೇಶಿಸುತ್ತದೆ, ಯಾವ ಉಪಕರಣಗಳನ್ನು ಕರೆಯಲಾಗುತ್ತದೆ, ಯಾವ ಫಲಿತಾಂಶಗಳು ಮರಳಿ ಬರುತ್ತವೆ ಮತ್ತು
  ಸಿಸ್ಟಂನಿಂದ ಏನು ಹೊರಹೋಗುತ್ತದೆ. ಪ್ರತಿ ನಿರ್ಧಾರವನ್ನು ಆಡಿಟ್-ಲಾಗ್ ಮಾಡಲಾಗುತ್ತದೆ.
- **ಡಿಫಾಲ್ಟ್ ನಿರಾಕರಣೆ** — ಯಾವುದೂ ಮೌನವಾಗಿ ಅನುಮತಿಸಲ್ಪಡುವುದಿಲ್ಲ. ವರ್ಗೀಕರಿಸದ ಉಪಕರಣಗಳು,
  ಏಕೀಕರಣಗಳು ಮತ್ತು ಡೇಟಾ ಮೂಲಗಳನ್ನು ಸ್ಪಷ್ಟವಾಗಿ ಕಾನ್ಫಿಗರ್ ಮಾಡದ ಹೊರತು ತಿರಸ್ಕರಿಸಲಾಗುತ್ತದೆ.
- **ಏಜೆಂಟ್ ಗುರುತು** — ನಿಮ್ಮ ಏಜೆಂಟ್‌ನ ಮಿಷನ್ SPINE.md ನಲ್ಲಿ ಇರುತ್ತದೆ, ಸಕ್ರಿಯ
  ನಡವಳಿಕೆಗಳು TRIGGER.md ನಲ್ಲಿ. Skills ಸರಳ ಫೋಲ್ಡರ್ ಸಂಪ್ರದಾಯಗಳ ಮೂಲಕ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ವಿಸ್ತರಿಸುತ್ತವೆ.
  The Reef ಮಾರ್ಕೆಟ್‌ಪ್ಲೇಸ್ ನಿಮಗೆ ಅವುಗಳನ್ನು ಕಂಡುಹಿಡಿಯಲು ಮತ್ತು ಹಂಚಿಕೊಳ್ಳಲು ಅನುಮತಿಸುತ್ತದೆ.
