# ಪ್ರಾರಂಭ ಮಾಡುವುದು

Triggerfish ಒಂದು ಸುರಕ್ಷಿತ, ಬಹು-ಚಾನೆಲ್ AI ಏಜೆಂಟ್ ಪ್ಲ್ಯಾಟ್‌ಫಾರ್ಮ್ ಆಗಿದ್ದು LLM ಪದರದ ಕೆಳಗೆ
ನಿರ್ಧಾರಾತ್ಮಕ ನೀತಿ ಜಾರಿಯನ್ನು ಒಳಗೊಂಡಿದೆ. ಇದು ನೀವು ಈಗಾಗಲೇ ಬಳಸುತ್ತಿರುವ ಪ್ರತಿ ಮೆಸೇಜಿಂಗ್ ಪ್ಲ್ಯಾಟ್‌ಫಾರ್ಮ್‌ಗೆ
ಸಂಪರ್ಕಿಸುತ್ತದೆ -- Telegram, Slack, Discord, WhatsApp, Email, WebChat ಮತ್ತು CLI --
AI ಬೈಪಾಸ್ ಮಾಡಲು, ಅತಿಕ್ರಮಿಸಲು ಅಥವಾ ಪ್ರಭಾವಿಸಲು ಸಾಧ್ಯವಿಲ್ಲದ ಡೇಟಾ ವರ್ಗೀಕರಣ ನಿಯಮಗಳನ್ನು ಜಾರಿಗೊಳಿಸುತ್ತಲೇ.

ನೀವು ನಿಜವಾದ ಭದ್ರತಾ ಖಾತರಿಗಳೊಂದಿಗೆ ಶಕ್ತಿಶಾಲಿ ವೈಯಕ್ತಿಕ AI ಸಹಾಯಕ ಬಯಸುವ ವ್ಯಕ್ತಿಯಾಗಿರಲಿ
ಅಥವಾ ಉದ್ಯೋಗಿ-ಏಜೆಂಟ್ ಸಂವಾದಗಳ ಉದ್ದಕ್ಕೂ ಆಡಿಟ್ ಮಾಡಬಹುದಾದ ನೀತಿ ಜಾರಿ ಅಗತ್ಯವಿರುವ ಸಂಸ್ಥೆಯಾಗಿರಲಿ,
Triggerfish ನಿಮಗೆ ಏಕೀಕರಣ ವ್ಯಾಪ್ತಿ ಮತ್ತು ವಿಶ್ವಾಸದಿಂದ AI ಬಳಸಲು ನಿಯಂತ್ರಣಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ.

Triggerfish ಮುಕ್ತ ಮೂಲ (Apache 2.0 ಪರವಾನಗಿ), ನಿಮ್ಮ ಯಂತ್ರದಲ್ಲಿ ಸ್ಥಳೀಯವಾಗಿ ಚಲಿಸುತ್ತದೆ,
ಮತ್ತು Anthropic Claude, OpenAI, Google Gemini, Ollama ಮೂಲಕ ಸ್ಥಳೀಯ ಮಾದರಿಗಳು ಮತ್ತು
OpenRouter ಸೇರಿದಂತೆ ಅನೇಕ LLM ಪ್ರದಾಯಕಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.
