### آپ کی ہمیں دی گئی personal information

***مختصراً:** ہم وہ personal information جمع کرتے ہیں جو آپ ہمیں فراہم کرتے ہیں۔*

**آپ کی فراہم کردہ Personal Information۔** ہم جو personal information جمع کرتے ہیں وہ ہمارے ساتھ آپ کے interactions کے context، آپ کی choices، اور آپ کے استعمال کردہ products اور features پر منحصر ہے۔ ہم جو personal information جمع کر سکتے ہیں اس میں یہ شامل ہو سکتا ہے:

- نام
- email addresses
- contact یا authentication data
- passwords

**Sensitive Information۔** ہم sensitive information process نہیں کرتے۔

**Payment Data۔** اگر آپ خریداری کرنے کا انتخاب کریں تو ہم آپ کی payment process کرنے کے لیے ضروری data جمع کر سکتے ہیں، جیسے آپ کا payment instrument number، اور آپ کے payment instrument سے منسلک security code۔ تمام payment data Stripe کے ذریعے handle اور store کی جاتی ہے۔ آپ ان کی privacy notice link(s) یہاں پا سکتے ہیں: [https://stripe.com/privacy](https://stripe.com/privacy)۔

***مختصراً:** ہم آپ کی personal information صرف تب process کرتے ہیں جب ہمیں یقین ہو کہ یہ ضروری ہے اور ہمارے پاس applicable law کے تحت ایسا کرنے کی valid legal وجہ (یعنی legal basis) ہے، جیسے آپ کی consent کے ساتھ، laws کی تعمیل کے لیے، آپ کو services فراہم کرنے کے لیے، contractual obligations پوری کرنے کے لیے، آپ کے حقوق کی حفاظت کے لیے، یا ہمارے legitimate business interests پورے کرنے کے لیے۔*

***اگر آپ EU یا UK میں ہیں تو یہ section آپ پر apply ہوتا ہے۔***

General Data Protection Regulation (GDPR) اور UK GDPR ہم سے یہ بتانے کا تقاضہ کرتے ہیں کہ ہم آپ کی personal information process کرنے کے لیے کن valid legal bases پر انحصار کرتے ہیں۔ جیسے، ہم آپ کی personal information process کرنے کے لیے درج ذیل legal bases پر انحصار کر سکتے ہیں:

- **Consent۔** ہم آپ کی معلومات process کر سکتے ہیں اگر آپ نے ہمیں مخصوص مقصد کے لیے اپنی personal information استعمال کرنے کی permission (یعنی consent) دی ہو۔ آپ کسی بھی وقت اپنی consent واپس لے سکتے ہیں۔ [اپنی consent واپس لینے](#withdrawing-your-consent) کے بارے میں مزید جانیں۔
- **Performance of a Contract۔**
- **Legitimate Interests۔**
- **Legal Obligations۔**
- **Vital Interests۔**

***اگر آپ Canada میں ہیں تو یہ section آپ پر apply ہوتا ہے۔***

ہم آپ کی معلومات process کر سکتے ہیں اگر آپ نے ہمیں specific permission (یعنی express consent) دی ہو یا ایسی situations میں جہاں آپ کی permission infer کی جا سکے (یعنی implied consent)۔ آپ کسی بھی وقت [اپنی consent واپس لے سکتے ہیں](#withdrawing-your-consent)۔
