# شروعات

Triggerfish ایک محفوظ، کثیر چینل AI ایجنٹ پلیٹ فارم ہے جس میں LLM کی تہہ کے نیچے
یقینی پالیسی نافذ کاری ہوتی ہے۔ یہ ہر اس پیغام رسانی پلیٹ فارم سے جڑتا ہے جو آپ
پہلے سے استعمال کرتے ہیں — Telegram، Slack، Discord، WhatsApp، Email، WebChat، اور
CLI — جبکہ ڈیٹا classification قوانین نافذ کرتا ہے جنہیں AI بائی پاس، اوور رائڈ،
یا متاثر نہیں کر سکتا۔

چاہے آپ ایک فرد ہوں جو حقیقی سیکیورٹی ضمانتوں کے ساتھ طاقتور ذاتی AI اسسٹنٹ چاہتے ہیں،
یا کوئی تنظیم جسے ملازم-ایجنٹ تعاملات میں قابل آڈٹ پالیسی نافذ کاری کی ضرورت ہے،
Triggerfish آپ کو اعتماد کے ساتھ AI استعمال کرنے کے لیے integration کی وسعت اور
کنٹرول دونوں فراہم کرتا ہے۔

Triggerfish اوپن سورس ہے (Apache 2.0 لائسنس یافتہ)، آپ کی مشین پر مقامی طور پر چلتا ہے،
اور متعدد LLM فراہم کنندگان کی حمایت کرتا ہے جن میں Anthropic Claude، OpenAI، Google
Gemini، Ollama کے ذریعے مقامی ماڈلز، اور OpenRouter شامل ہیں۔
