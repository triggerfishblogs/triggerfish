# தொடங்குங்கள்

Triggerfish என்பது LLM அடுக்கிற்கு கீழே நிர்ணயவாத கொள்கை அமலாக்கத்துடன் கூடிய பாதுகாப்பான, பல-சேனல் AI agent தளம். இது நீங்கள் ஏற்கனவே பயன்படுத்தும் ஒவ்வொரு messaging தளத்திலும் இணைகிறது -- Telegram, Slack, Discord, WhatsApp, Email, WebChat மற்றும் CLI -- AI க்கு bypass, override அல்லது தாக்கம் செய்ய முடியாத data வகைப்படுத்தல் விதிகளை அமல்படுத்தும்போது.

உண்மையான பாதுகாப்பு உத்தரவாதங்களுடன் சக்திவாய்ந்த தனிப்பட்ட AI உதவியாளரை விரும்பும் ஒரு நபராக இருந்தாலும், அல்லது employee-agent தொடர்புகளில் தணிக்கை-யோக்கியமான கொள்கை அமலாக்கம் தேவைப்படும் ஒரு நிறுவனமாக இருந்தாலும், Triggerfish உங்களுக்கு integration அகலத்தையும் நம்பிக்கையுடன் AI பயன்படுத்துவதற்கான கட்டுப்பாடுகளையும் வழங்குகிறது.

Triggerfish திறந்த மூல (Apache 2.0 உரிமம்), உங்கள் கணினியில் உள்ளூரில் இயங்குகிறது, மற்றும் Anthropic Claude, OpenAI, Google Gemini, Ollama மூலம் உள்ளூர் மாதிரிகள் மற்றும் OpenRouter உள்பட பல LLM வழங்குநர்களை ஆதரிக்கிறது.
