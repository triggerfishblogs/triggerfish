## இன்னும் Stuck ஆகிறீர்களா?

மேலே உள்ள docs உங்கள் problem solve செய்யவில்லையென்றால்:

1. யாரேனும் ஏற்கனவே report செய்தார்களா என்று [GitHub Issues](https://github.com/greghavens/triggerfish/issues) இல் **existing issues தேடவும்**
2. [GitHub Discussions](https://github.com/greghavens/triggerfish/discussions) இல் **community கேளுங்கள்**
3. [Issue filing guide](/ta-IN/support/guides/filing-issues) பின்பற்றி **புதிய issue file செய்யுங்கள்**
