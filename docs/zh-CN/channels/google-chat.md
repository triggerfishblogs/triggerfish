# Google Chat

<ComingSoon />

将您的 Triggerfish 智能体连接到 Google Chat，让使用 Google Workspace 的团队可以直接从聊天界面与其交互。适配器将使用 Google Chat API 配合服务账户或 OAuth 凭据。

## 计划功能

- 私聊和聊天室（空间）支持
- 通过 Google Workspace 目录进行所有者验证
- 输入指示
- 长响应的消息分块
- 与其他渠道一致的分级执行

## 配置（计划中）

```yaml
channels:
  google-chat:
    classification: INTERNAL
```

有关现有的涵盖 Gmail、日历、任务、云端硬盘和表格的 Google 集成，请参阅 [Google Workspace](/zh-CN/integrations/google-workspace)。
