# 入门指南

Triggerfish 是一个安全的多渠道 AI 智能体平台，在 LLM 层以下实现确定性策略执行。它连接到您已经使用的每个消息平台 —— Telegram、Slack、Discord、WhatsApp、Email、WebChat 和 CLI —— 同时执行 AI 无法绕过、覆盖或影响的数据分类规则。

无论您是想要一个具有真正安全保障的强大个人 AI 助手的个人用户，还是需要跨员工-智能体交互的可审计策略执行的组织，Triggerfish 都为您提供了集成广度和使用 AI 的信心控制。

Triggerfish 是开源的（Apache 2.0 许可证），在您的机器上本地运行，支持多种 LLM 提供商，包括 Anthropic Claude、OpenAI、Google Gemini、通过 Ollama 使用的本地模型和 OpenRouter。
