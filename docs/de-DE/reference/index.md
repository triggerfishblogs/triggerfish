# Referenz

Dieser Abschnitt bietet technisches Referenzmaterial fuer die Konfiguration und Erweiterung von Triggerfish.

## [Konfigurationsschema](./config-yaml)

Vollstaendige Referenz fuer `triggerfish.yaml` -- jeder Abschnitt, jeder Schluessel, mit einem annotierten Beispiel. Behandelt Models, Channels, Klassifizierung, Policy, MCP-Server, Scheduler und Benachrichtigungen.

## [Wichtige Interfaces](./interfaces)

TypeScript-Interface-Referenz fuer Entwickler, die Integrationen, benutzerdefinierte Channel-Adapter, LLM-Provider oder Storage-Backends erstellen. Umfasst `StorageProvider`, `ChannelAdapter`, `LlmProvider`, `NotificationService`, Hook-Typen, Session-Status und das `Result<T, E>`-Muster.

## [Glossar](./glossary)

Definitionen fuer jeden Triggerfish-spezifischen Begriff, von Buoy bis Write-Down.
