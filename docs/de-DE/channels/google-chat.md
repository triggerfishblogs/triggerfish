# Google Chat

<ComingSoon />

Verbinden Sie Ihren Triggerfish-Agenten mit Google Chat, damit Teams, die Google Workspace nutzen, direkt aus ihrer Chat-Oberflaeche mit ihm interagieren koennen. Der Adapter wird die Google Chat API mit Service-Account- oder OAuth-Anmeldedaten verwenden.

## Geplante Features

- Direktnachrichten- und Space- (Raum-) Unterstuetzung
- Eigentuemer-Verifizierung ueber Google-Workspace-Verzeichnis
- Tipp-Indikatoren
- Nachrichtenaufteilung fuer lange Antworten
- Klassifizierungsdurchsetzung konsistent mit anderen Kanaelen

## Konfiguration (geplant)

```yaml
channels:
  google-chat:
    classification: INTERNAL
```

Siehe [Google Workspace](/de-DE/integrations/google-workspace) fuer die bestehende Google-Integration, die Gmail, Kalender, Aufgaben, Drive und Tabellen abdeckt.
