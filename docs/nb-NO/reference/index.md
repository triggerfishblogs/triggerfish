# Referanse

Denne seksjonen inneholder teknisk referansemateriale for konfigurering og
utvidelse av Triggerfish.

## [Konfigurasjonsskjema](./config-yaml)

Fullstendig referanse for `triggerfish.yaml` — hver seksjon, hver nøkkel, med
et kommentert eksempel. Dekker modeller, kanaler, klassifisering, policy,
MCP-servere, scheduler og varsler.

## [Viktige grensesnitt](./interfaces)

TypeScript-grensesnitteferanse for utviklere som bygger integrasjoner, egendefinerte
kanaladaptere, LLM-leverandører eller lagringsservere. Inkluderer
`StorageProvider`, `ChannelAdapter`, `LlmProvider`, `NotificationService`,
hook-typer, sesjonstilstand og `Result<T, E>`-mønsteret.

## [Ordliste](./glossary)

Definisjoner for alle Triggerfish-spesifikke termer, fra Buoy til Write-Down.
