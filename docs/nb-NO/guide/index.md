# Kom i gang

Triggerfish er en sikker, flerkanal AI-agentplattform med deterministisk policy-håndhevelse under LLM-laget. Den kobler seg til alle meldingsplattformer du allerede bruker — Telegram, Slack, Discord, WhatsApp, e-post, WebChat og CLI — og håndhever dataklassifiseringsregler som KI-en ikke kan omgå, overstyre eller påvirke.

Enten du er en enkeltperson som ønsker en kraftig personlig AI-assistent med reelle sikkerhetsgarantier, eller en organisasjon som trenger revisjonsdyktig policy-håndhevelse på tvers av ansatt-agent-interaksjoner — Triggerfish gir deg både integrasjonsbredde og kontrollene du trenger for å bruke KI med trygghet.

Triggerfish er åpen kildekode (lisensiert under Apache 2.0), kjører lokalt på maskinen din og støtter flere LLM-leverandører, inkludert Anthropic Claude, OpenAI, Google Gemini, lokale modeller via Ollama og OpenRouter.
