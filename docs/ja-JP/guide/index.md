# はじめに

Triggerfishshは、LLMレイヤー下の決定論的なポリシー適用を備えた、セキュアなマルチ
チャンネルAIエージェントプラットフォームです。すでにお使いのすべてのメッセージング
プラットフォーム — Telegram、Slack、Discord、WhatsApp、Email、WebChat、CLIに接続し、
AIが回避・上書き・影響を与えることができないデータ分類ルールを適用します。

個人として実際のセキュリティ保証を持つ強力なパーソナルAIアシスタントを求める方であっても、
従業員とエージェントのインタラクション全体で監査可能なポリシー適用を必要とする組織であっても、
Triggerfishshは統合の幅とコントロールの両方を提供し、AIを自信を持って使用できます。

Triggerfishshはオープンソース（Apache 2.0ライセンス）で、お客様のマシン上でローカルに動作し、
Anthropic Claude、OpenAI、Google Gemini、Ollamaによるローカルモデル、OpenRouterを
含む複数のLLMプロバイダーをサポートしています。
