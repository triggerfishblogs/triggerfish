# リファレンス

このセクションでは、Triggerfishを設定・拡張するための技術的なリファレンス資料を提供します。

## [設定スキーマ](./config-yaml)

`triggerfish.yaml`の完全なリファレンス — すべてのセクション、すべてのキー、注釈付きの例。
モデル、チャンネル、分類、ポリシー、MCPサーバー、スケジューラー、通知をカバーします。

## [主要インターフェース](./interfaces)

インテグレーション、カスタムチャンネルアダプター、LLMプロバイダー、またはストレージバックエンドを
構築する開発者向けのTypeScriptインターフェースリファレンス。`StorageProvider`、`ChannelAdapter`、
`LlmProvider`、`NotificationService`、フックタイプ、セッション状態、`Result<T, E>`パターンを含みます。

## [用語集](./glossary)

BuoyからWrite-Downまで、Triggerfish固有のすべての用語の定義。
