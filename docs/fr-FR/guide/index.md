# Pour commencer

Triggerfish est une plateforme d'agent IA sécurisée et multicanale avec une
application déterministe des politiques en dessous de la couche LLM. Elle se
connecte à toutes les plateformes de messagerie que vous utilisez déjà --
Telegram, Slack, Discord, WhatsApp, Email, WebChat et le CLI -- tout en
appliquant des règles de classification des données que l'IA ne peut ni
contourner, ni outrepasser, ni influencer.

Que vous soyez un individu souhaitant un assistant IA personnel puissant avec de
véritables garanties de sécurité, ou une organisation nécessitant une
application auditable des politiques pour les interactions employé·e-agent,
Triggerfish vous offre à la fois la couverture d'intégration et les contrôles
pour utiliser l'IA en toute confiance.

Triggerfish est open source (licence Apache 2.0), s'exécute localement sur votre
machine et prend en charge plusieurs fournisseurs LLM, notamment Anthropic
Claude, OpenAI, Google Gemini, les modèles locaux via Ollama et OpenRouter.
