# Reference

Cette section fournit la documentation technique de reference pour configurer et etendre
Triggerfish.

## [Schema de configuration](./config-yaml)

Reference complete de `triggerfish.yaml` -- chaque section, chaque cle, avec un
exemple annote. Couvre les modeles, canaux, classification, politique, serveurs MCP,
planificateur et notifications.

## [Interfaces cles](./interfaces)

Reference des interfaces TypeScript pour les developpeurs construisant des integrations,
des adaptateurs de canaux personnalises, des fournisseurs LLM ou des backends de stockage. Inclut
`StorageProvider`, `ChannelAdapter`, `LlmProvider`, `NotificationService`, les types de
hooks, l'etat de session et le patron `Result<T, E>`.

## [Glossaire](./glossary)

Definitions de chaque terme specifique a Triggerfish, de Buoy a Write-Down.
