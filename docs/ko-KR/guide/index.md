# 시작하기

Triggerfish는 LLM 계층 아래에서 결정론적 정책 시행을 수행하는 보안 멀티채널 AI 에이전트 플랫폼입니다. Telegram, Slack, Discord, WhatsApp, Email, WebChat, CLI 등 이미 사용하고 계신 모든 메시징 플랫폼에 연결하면서, AI가 우회하거나 무시하거나 영향을 줄 수 없는 데이터 분류 규칙을 시행합니다.

강력한 개인 AI 비서와 실질적인 보안 보증을 원하는 개인이든, 직원-에이전트 상호작용 전반에 감사 가능한 정책 시행이 필요한 조직이든, Triggerfish는 통합의 폭과 제어 기능을 모두 제공하여 자신 있게 AI를 사용할 수 있도록 합니다.

Triggerfish는 오픈 소스(Apache 2.0 라이선스)이며, 로컬 머신에서 실행되고, Anthropic Claude, OpenAI, Google Gemini, Ollama를 통한 로컬 모델, OpenRouter 등 다양한 LLM 제공자를 지원합니다.
