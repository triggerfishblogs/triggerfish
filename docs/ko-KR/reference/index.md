# 레퍼런스

이 섹션은 Triggerfish를 구성하고 확장하기 위한 기술 참조 자료를 제공합니다.

## [구성 스키마](./config-yaml)

`triggerfish.yaml`의 전체 참조 -- 모든 섹션, 모든 키, 주석이 달린 예시 포함. 모델, 채널, 분류, 정책, MCP 서버, 스케줄러, 알림을 다룹니다.

## [주요 인터페이스](./interfaces)

통합, 사용자 정의 채널 어댑터, LLM 제공자 또는 스토리지 백엔드를 구축하는 개발자를 위한 TypeScript 인터페이스 참조. `StorageProvider`, `ChannelAdapter`, `LlmProvider`, `NotificationService`, hook 타입, 세션 상태, `Result<T, E>` 패턴을 포함합니다.

## [용어집](./glossary)

Buoy부터 Write-Down까지 모든 Triggerfish 전용 용어의 정의.
