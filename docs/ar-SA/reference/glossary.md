# المسرد

تعريفات للمصطلحات الخاصة بـ Triggerfish.

| المصطلح | التعريف |
|---------|---------|
| **A2UI** | بروتوكول Agent-to-UI. يدفع التحديثات في الوقت الفعلي من الوكيل لواجهات المستخدم المتصلة. |
| **Buoy** | تطبيق مرافق يوفر قدرات الجهاز (الكاميرا، الموقع، إلخ.) |
| **Classification** | مستوى حساسية البيانات (RESTRICTED > CONFIDENTIAL > INTERNAL > PUBLIC) |
| **Dive** | معالج الإعداد الأول (`triggerfish dive`). يُنشئ SPINE.md و triggerfish.yaml |
| **Gateway** | مستوى التحكم المركزي. خدمة محلية طويلة التشغيل تدير الجلسات والقنوات والأدوات |
| **Hook** | نقطة تنفيذ حتمية في تدفق البيانات |
| **MCP** | Model Context Protocol. بروتوكول لتوصيل أدوات وبيانات خارجية |
| **No Write-Down** | قاعدة تمنع تدفق البيانات لمستوى تصنيف أقل |
| **Patrol** | أمر فحص صحة تشخيصي (`triggerfish patrol`) |
| **Plugin** | كود خارجي يعمل داخل عزل مزدوج (Deno + WASM) |
| **Reef, The** | سوق المهارات لاكتشاف وتثبيت ونشر المهارات |
| **Ripple** | مؤشرات الكتابة وحالة الاتصال عبر القنوات |
| **SDK** | Plugin SDK. واجهة للإضافات الخارجية |
| **Session** | الوحدة الأساسية لحالة المحادثة مع تتبع taint مستقل |
| **Skill** | مجلد يحتوي SKILL.md يمنح الوكيل قدرات جديدة |
| **SPINE.md** | ملف هوية ومهمة الوكيل (أساس prompt النظام) |
| **SSRF** | تزوير الطلبات من جانب الخادم. يُمنع بقائمة حظر IP مُرمزة |
| **Taint** | تصنيف على مستوى الجلسة يتصاعد عند الوصول لبيانات مصنفة. لا يمكنه الانخفاض |
| **Tide Pool** | مساحة عمل مرئية مُعرضة عبر بروتوكول A2UI |
| **TRIGGER.md** | تعريف السلوك الاستباقي للوكيل |
| **Trigger** | إيقاظ وكيل دوري للسلوك الاستباقي المستقل |
| **Write-Down** | تدفق محظور للبيانات لمستوى تصنيف أقل |
