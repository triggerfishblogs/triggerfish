# المرجع

يوفر هذا القسم مواد مرجعية تقنية لتكوين Triggerfish وتوسيعه.

## [مخطط التكوين](./config-yaml)

مرجع كامل لـ `triggerfish.yaml` -- كل قسم وكل مفتاح، مع مثال مُعلق. يغطي
النماذج والقنوات والتصنيف والسياسات وخوادم MCP والمجدول والإشعارات.

## [الواجهات الرئيسية](./interfaces)

مرجع واجهات TypeScript للمطورين الذين يبنون تكاملات أو محولات قنوات مخصصة أو
مزودي LLM أو واجهات تخزين خلفية.

## [المسرد](./glossary)

تعريفات لكل مصطلح خاص بـ Triggerfish، من Buoy إلى Write-Down.
