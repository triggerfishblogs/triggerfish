# إدارة الجلسات

يمكن للوكيل فحص الجلسات والتواصل معها وإنشاء جلسات جديدة. تمكّن هذه الأدوات
سير العمل عبر الجلسات وتفويض المهام الخلفية والمراسلة عبر القنوات -- كل ذلك
تحت تنفيذ منع الكتابة للأسفل.

## الأدوات

### `sessions_list`

سرد الجلسات النشطة مع مرشحات اختيارية. لا تغيير في taint.

### `sessions_history`

استرجاع نص جلسة. Taint يرث من الجلسة المُشار إليها.

### `sessions_send`

إرسال رسالة لجلسة أخرى. يخضع لفحص منع الكتابة للأسفل.

### `sessions_spawn`

إنشاء جلسة مهمة خلفية. تبدأ بـ taint `PUBLIC`.

### `session_status`

فحص حالة الجلسة الحالية: النموذج، عدد الرموز، التكلفة، مستوى taint.
