# التوجيه متعدد الوكلاء

يدعم Triggerfish توجيه القنوات والحسابات وجهات الاتصال المختلفة إلى وكلاء معزولين
منفصلين، كل منها بمساحة عمله وجلساته وشخصيته وسقف تصنيفه.

## كيف يعمل

وجّه وكلاء مختلفين لقنوات مختلفة:

- Slack العمل يذهب لوكيل واحد
- WhatsApp الشخصي يذهب لوكيل آخر
- كل وكيل لديه SPINE.md خاص ومساحة عمل ومهارات وتتبع taint

## تكوين الوكيل

كل وكيل لديه:

- `SPINE.md` خاص في `~/.triggerfish/workspace/<agent-id>/`
- `TRIGGER.md` اختياري خاص
- جلسات معزولة بتتبع taint مستقل
- سقف تصنيف يحد من البيانات التي يمكنه الوصول إليها

## عزل الأمان

- الوكلاء لا يمكنهم الوصول لجلسات أو مساحات عمل بعضهم
- الاتصال بين الوكلاء يخضع لسلاسل التفويض ومنع الكتابة للأسفل
- كل وكيل لديه سقف تصنيف مستقل
