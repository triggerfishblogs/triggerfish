# نظرة عامة على الميزات

بالإضافة إلى [نموذج الأمان](/ar-SA/security/) و[دعم القنوات](/ar-SA/channels/)،
يوفر Triggerfish قدرات تمد وكيل الذكاء الاصطناعي إلى ما وراء السؤال والجواب:
المهام المجدولة، والذاكرة الدائمة، والوصول للويب، وإدخال الصوت، وتجاوز فشل
النماذج المتعددة.

## السلوك الاستباقي

### [Cron و Triggers](./cron-and-triggers)

جدول مهام متكررة بتعبيرات cron قياسية وعرّف سلوك مراقبة استباقي عبر `TRIGGER.md`.

### [الإشعارات](./notifications)

خدمة تسليم إشعارات توجه الرسائل عبر جميع القنوات المتصلة مع مستويات أولوية
وترتيب في قائمة الانتظار عند عدم الاتصال وإزالة التكرار.

## أدوات الوكيل

### [البحث على الويب والجلب](./web-search)

البحث على الويب وجلب محتوى الصفحات مع منع SSRF وتنفيذ السياسات.

### [الذاكرة الدائمة](./memory)

ذاكرة عبر الجلسات مع بوابة تصنيف. يحفظ الوكيل ويستدعي الحقائق والتفضيلات
والسياق عبر المحادثات.

### [تحليل الصور والرؤية](./image-vision)

الصق الصور من الحافظة (Ctrl+V في CLI) وحلل ملفات الصور على القرص.

### [استكشاف قاعدة الكود](./explore)

فهم منظم لقاعدة الكود عبر وكلاء فرعيين متوازيين.

### [إدارة الجلسات](./sessions)

افحص الجلسات وتواصل معها وأنشئ جلسات جديدة مع تنفيذ منع الكتابة للأسفل.

### [وضع الخطة وتتبع المهام](./planning)

تخطيط منظم قبل التنفيذ (وضع الخطة) وتتبع مهام دائم عبر الجلسات.

### [نظام الملفات والصدفة](./filesystem)

قراءة وكتابة وبحث وتنفيذ أوامر مع تحديد نطاق مساحة العمل.

### [الوكلاء الفرعيون ومهام LLM](./subagents)

تفويض العمل لوكلاء فرعيين مستقلين أو تشغيل prompts LLM معزولة.

### [فرق الوكلاء](./agent-teams)

إنشاء فرق دائمة من وكلاء متعاونين بأدوار متخصصة.

## التفاعل الغني

### [خط أنابيب الصوت](./voice)

دعم كلام كامل مع مزودي STT و TTS قابلين للتكوين.

### [Tide Pool / A2UI](./tidepool)

مساحة عمل مرئية مدفوعة بالوكيل حيث يعرض Triggerfish محتوى تفاعلياً.

## متعدد الوكلاء ومتعدد النماذج

### [التوجيه متعدد الوكلاء](./multi-agent)

وجّه القنوات والحسابات المختلفة إلى وكلاء معزولين منفصلين.

### [مزودو LLM وتجاوز الفشل](./model-failover)

اتصل بـ Anthropic, OpenAI, Google, النماذج المحلية (Ollama), أو OpenRouter مع
سلاسل تجاوز فشل.

### [تحديد المعدل](./rate-limiting)

محدد معدل بنافذة منزلقة يمنع الوصول لحدود API مزود LLM.

## العمليات

### [التسجيل المُهيكل](./logging)

تسجيل مُهيكل موحد مع مستويات خطورة وتدوير ملفات ومخرجات مزدوجة.

::: info جميع الميزات تتكامل مع نموذج الأمان الأساسي. مهام cron تحترم سقوف
التصنيف. إدخال الصوت يحمل taint. محتوى Tide Pool يمر عبر hook PRE_OUTPUT.
التوجيه متعدد الوكلاء يفرض عزل الجلسات. لا ميزة تتجاوز طبقة السياسات. :::
