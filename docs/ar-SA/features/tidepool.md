# Tide Pool / A2UI

Tide Pool هو مساحة عمل مرئية مدفوعة بالوكيل حيث يعرض Triggerfish محتوى تفاعلياً:
لوحات معلومات، ورسوم بيانية، ونماذج، ومعاينات كود، ووسائط غنية. على عكس
المحادثة الخطية، Tide Pool هو لوحة يُحدثها الوكيل في الوقت الفعلي.

## بروتوكول A2UI

بروتوكول Agent-to-UI (A2UI) يدفع التحديثات في الوقت الفعلي من الوكيل للعملاء
المتصلين عبر WebSocket.

## الأدوات

### `tidepool_canvas`

إنشاء أو تحديث لوحة Tide Pool بمحتوى HTML/CSS/JS.

### `tidepool_clear`

مسح محتوى Tide Pool الحالي.

## الأمان

محتوى Tide Pool يمر عبر hook `PRE_OUTPUT` قبل التسليم. المحتوى يخضع لنفس قواعد
التصنيف كأي مخرج آخر.

## الوصول

Tide Pool يُستضاف بواسطة Gateway على المنفذ المُكوّن (افتراضي 18790).
