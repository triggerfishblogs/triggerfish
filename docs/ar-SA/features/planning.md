# وضع الخطة وتتبع المهام

يوفر Triggerfish أداتين متكاملتين للعمل المُنظم: **وضع الخطة** للتخطيط المعقد
للتنفيذ، و**تتبع المهام** لإدارة المهام عبر الجلسات.

## وضع الخطة

وضع الخطة يقيد الوكيل بالاستكشاف للقراءة فقط حتى يوافق المستخدم على الخطة.

### `plan_start`

ادخل وضع الخطة. الوكيل يمكنه القراءة والبحث والاستكشاف لكن لا يمكنه الكتابة
أو التنفيذ.

### `plan_finish`

اخرج من وضع الخطة وقدم الخطة للموافقة.

## تتبع المهام

### `todo_add`

أضف عنصراً لقائمة المهام.

### `todo_list`

اسرد جميع المهام.

### `todo_complete`

أكمل عنصر مهمة بالمعرف.

### `todo_clear`

احذف جميع المهام المكتملة.

## الاستمرارية

المهام تُخزن عبر `StorageProvider` وتستمر عبر الجلسات وإعادات التشغيل.
