# البدء

Triggerfish هو منصة وكلاء ذكاء اصطناعي آمنة ومتعددة القنوات مع تنفيذ سياسات
حتمي أسفل طبقة LLM. يتصل بكل منصة مراسلة تستخدمها بالفعل -- Telegram, Slack,
Discord, WhatsApp, البريد الإلكتروني, WebChat, و CLI -- مع تنفيذ قواعد تصنيف
البيانات التي لا يستطيع الذكاء الاصطناعي تجاوزها أو تعديلها أو التأثير عليها.

سواء كنت فرداً يريد مساعداً ذكاء اصطناعي شخصياً قوياً مع ضمانات أمنية حقيقية،
أو مؤسسة تحتاج تنفيذ سياسات قابل للتدقيق عبر تفاعلات الموظفين مع الوكيل،
يمنحك Triggerfish عرض التكامل والضوابط لاستخدام الذكاء الاصطناعي بثقة.

Triggerfish مفتوح المصدر (مرخص Apache 2.0)، يعمل محلياً على جهازك، ويدعم مزودي
LLM متعددين بما في ذلك Anthropic Claude, OpenAI, Google Gemini, النماذج المحلية
عبر Ollama, و OpenRouter.
