# قاعدة منع الكتابة للأسفل

الثابت الأمني الأهم في Triggerfish:

::: danger يمكن للبيانات أن تتدفق فقط إلى تصنيف **مساوٍ أو أعلى**. لا توجد
استثناءات. لا تجاوزات. لا تكوينات تُعطلها. :::

## كيف تعمل

```
RESTRICTED ← يمكن الإرسال إلى: RESTRICTED فقط
CONFIDENTIAL ← يمكن الإرسال إلى: CONFIDENTIAL, RESTRICTED
INTERNAL ← يمكن الإرسال إلى: INTERNAL, CONFIDENTIAL, RESTRICTED
PUBLIC ← يمكن الإرسال إلى: أي مكان
```

## أين تُنفذ

- **PRE_OUTPUT hook** -- يفحص كل مخرج قبل التسليم
- **sessions_send** -- الاتصال بين الجلسات
- **الإشعارات** -- محتوى الإشعارات
- **تفويض الوكيل** -- سلاسل الوكيل لوكيل
- **الذاكرة** -- كتابة وقراءة الذاكرة عبر الجلسات

## لماذا هي قاعدة ثابتة

هذه ليست قاعدة قابلة للتكوين. إنها مُرمزة في محرك السياسات ولا يمكن لأي مسؤول
أو مستخدم أو تكوين تعطيلها. إذا كانت قابلة للتكوين، فهي ليست ثابتة.
