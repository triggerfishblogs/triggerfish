# تسجيل التدقيق

كل قرار متعلق بالأمان يُسجل مع سياق كامل. تسجيل التدقيق لا يمكن تعطيله --
إنه قاعدة ثابتة في تسلسل السياسات الهرمي.

## ما يُسجل

- جميع طلبات الإجراءات (المسموحة والمرفوضة)
- قرارات التصنيف
- تغييرات taint الجلسة
- أحداث مصادقة القناة
- تقييمات قواعد السياسات
- سجلات النسب
- قرارات MCP Gateway
- استدعاءات وكيل لوكيل

## شكل السجل

```json
{
  "timestamp": "2025-01-29T10:23:45Z",
  "user_id": "user_123",
  "session_id": "sess_456",
  "action": "slack.postMessage",
  "session_taint": "CONFIDENTIAL",
  "target_classification": "PUBLIC",
  "decision": "DENIED",
  "reason": "classification_violation",
  "hook": "PRE_OUTPUT",
  "policy_rules_evaluated": ["rule_001", "rule_002"]
}
```

## الاحتفاظ

سجلات التدقيق تُحفظ لمدة سنة واحدة افتراضياً (قابل للتكوين في عمليات النشر
المؤسسية).

::: info لا يمكن لأي مسؤول منظمة إيقاف التسجيل لإجراءاته الخاصة. يمكن لعمليات
النشر المؤسسية تفعيل تسجيل المحتوى الكامل اختيارياً. :::
