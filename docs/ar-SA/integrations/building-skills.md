# بناء المهارات

المهارات توسع قدرات وكيلك بمعرفة ومهام متخصصة. ملف `SKILL.md` في أي مجلد
يُعلم Triggerfish أن هذا المجلد يحتوي على مهارة.

## بنية المهارة

```
my-skill/
├── SKILL.md          # تعريف المهارة (مطلوب)
├── templates/        # قوالب اختيارية
└── examples/         # أمثلة اختيارية
```

## SKILL.md

ملف SKILL.md يحدد:

- **الاسم والوصف**: ما تفعله المهارة
- **المُشغلات**: متى يجب تفعيل المهارة
- **التعليمات**: كيف يجب على الوكيل استخدام المهارة
- **الأدوات**: أي أدوات خاصة تحتاجها المهارة

## أنواع المهارات

| النوع     | الموقع                                        | الوصف                        |
| --------- | --------------------------------------------- | ---------------------------- |
| مُدمجة    | `src/skills/`                                 | تُشحن مع Triggerfish         |
| مُدارة    | `~/.triggerfish/skills/`                       | مُثبتة من The Reef           |
| مساحة عمل | `~/.triggerfish/workspace/<agent>/skills/`     | مُنشأة بواسطة الوكيل         |

## النشر

```bash
triggerfish skill publish
```

ينشر مهارتك في The Reef للآخرين لاستخدامها.
