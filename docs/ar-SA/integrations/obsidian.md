# Obsidian

يتكامل Triggerfish مع Obsidian لتدوين الملاحظات والبحث فيها.

## الإعداد

فعّل plugin Obsidian أثناء `triggerfish dive` أو أضفه يدوياً للتكوين.

## الأدوات

| الأداة             | الوصف                              |
| ------------------ | ---------------------------------- |
| `obsidian_search`  | البحث في ملاحظات Obsidian           |
| `obsidian_read`    | قراءة ملاحظة محددة                  |
| `obsidian_create`  | إنشاء ملاحظة جديدة                 |
| `obsidian_update`  | تحديث ملاحظة موجودة                |

## التصنيف

يُصنف الوصول لملاحظات Obsidian كـ `INTERNAL` افتراضياً. يمكنك تعديل هذا في
التكوين.
