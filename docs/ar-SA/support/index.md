# مركز الدعم

احصل على المساعدة في تثبيت Triggerfish وتكوينه والعمليات اليومية.

## روابط سريعة

- **هل هناك شيء معطّل الآن؟** ابدأ بـ[دليل استكشاف الأخطاء وإصلاحها](/ar-SA/support/troubleshooting/)
- **هل تحتاج إلى البحث عن خطأ؟** راجع [مرجع الأخطاء](/ar-SA/support/troubleshooting/error-reference)
- **هل تريد الإبلاغ عن خلل؟** اقرأ [كيفية تقديم بلاغ جيد](/ar-SA/support/guides/filing-issues) أولاً
- **هل تقوم بالترقية أو الترحيل؟** تحقق من [قاعدة المعرفة](#قاعدة-المعرفة)

## موارد الخدمة الذاتية

### استكشاف الأخطاء وإصلاحها

أدلة خطوة بخطوة لتشخيص المشكلات الشائعة وإصلاحها، منظمة حسب المجال:

| المجال | يغطي |
|------|--------|
| [التثبيت](/ar-SA/support/troubleshooting/installation) | فشل التثبيت، أخطاء الأذونات، الإعداد حسب المنصة |
| [الخدمة الخلفية](/ar-SA/support/troubleshooting/daemon) | مشاكل البدء/الإيقاف، إدارة الخدمة، تعارضات المنافذ |
| [التكوين](/ar-SA/support/troubleshooting/configuration) | تحليل YAML، أخطاء التحقق، مراجع الأسرار |
| [القنوات](/ar-SA/support/troubleshooting/channels) | Telegram، Slack، Discord، WhatsApp، Signal، Email، WebChat |
| [مزودو LLM](/ar-SA/support/troubleshooting/providers) | أخطاء مفاتيح API، النموذج غير موجود، فشل البث، التبديل التلقائي |
| [التكاملات](/ar-SA/support/troubleshooting/integrations) | Google، GitHub، Notion، CalDAV، خوادم MCP |
| [أتمتة المتصفح](/ar-SA/support/troubleshooting/browser) | اكتشاف Chrome، فشل التشغيل، Flatpak، التنقل |
| [الأمان والتصنيف](/ar-SA/support/troubleshooting/security) | تصعيد التلوث، حظر الكتابة للأسفل، SSRF، رفض السياسات |
| [الأسرار وبيانات الاعتماد](/ar-SA/support/troubleshooting/secrets) | واجهات سلسلة المفاتيح، أخطاء الأذونات، مخزن الملفات المشفر |
| [مرجع الأخطاء](/ar-SA/support/troubleshooting/error-reference) | فهرس قابل للبحث لجميع رسائل الخطأ |

## هل ما زلت عالقاً؟

إذا لم تحل الوثائق أعلاه مشكلتك:

1. **ابحث في البلاغات الموجودة** على [GitHub Issues](https://github.com/greghavens/triggerfish/issues) لمعرفة ما إذا كان شخص ما قد أبلغ عنها بالفعل
2. **اسأل المجتمع** في [GitHub Discussions](https://github.com/greghavens/triggerfish/discussions)
3. **قدّم بلاغاً جديداً** باتباع [دليل تقديم البلاغات](/ar-SA/support/guides/filing-issues)
