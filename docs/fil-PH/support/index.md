## Hindi Pa Rin Naayos?

Kung hindi nalutas ng docs sa itaas ang iyong problema:

1. **Hanapin ang mga existing issues** sa [GitHub Issues](https://github.com/greghavens/triggerfish/issues) para tingnan kung may nag-report na
2. **Magtanong sa community** sa [GitHub Discussions](https://github.com/greghavens/triggerfish/discussions)
3. **Mag-file ng bagong issue** gamit ang [issue filing guide](/fil-PH/support/guides/filing-issues)
