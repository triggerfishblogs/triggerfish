# Reference

Nagbibigay ang section na ito ng technical reference material para sa pag-configure at pag-extend ng Triggerfish.

## [Config Schema](./config-yaml)

Buong reference para sa `triggerfish.yaml` -- bawat section, bawat key, na may annotated example. Sinasaklaw ang models, channels, classification, policy, MCP servers, scheduler, at notifications.

## [Mga Key Interface](./interfaces)

TypeScript interface reference para sa mga developer na nagbu-build ng integrations, custom channel adapters, LLM providers, o storage backends. Kasama ang `StorageProvider`, `ChannelAdapter`, `LlmProvider`, `NotificationService`, hook types, session state, at ang `Result<T, E>` pattern.

## [Glossary](./glossary)

Mga depinisyon para sa bawat Triggerfish-specific term, mula Buoy hanggang Write-Down.
