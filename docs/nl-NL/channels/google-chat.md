# Google Chat

<ComingSoon />

Verbind uw Triggerfish-agent met Google Chat zodat teams die Google Workspace gebruiken er direct vanuit hun chatinterface mee kunnen communiceren. De adapter gebruikt de Google Chat API met serviceaccount- of OAuth-inloggegevens.

## Geplande functies

- Ondersteuning voor directe berichten en spaces (ruimten)
- Eigenaarverificatie via Google Workspace-directory
- Typaanduidingen
- Berichtopsplitsing voor lange antwoorden
- Classificatiehandhaving consistent met andere kanalen

## Configuratie (gepland)

```yaml
channels:
  google-chat:
    classification: INTERNAL
```

Zie [Google Workspace](/nl-NL/integrations/google-workspace) voor de bestaande Google-integratie die Gmail, Agenda, Taken, Drive en Spreadsheets omvat.
