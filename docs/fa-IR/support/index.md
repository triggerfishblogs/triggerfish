# مرکز پشتیبانی

احصل روی المساعدة در تثبيت Triggerfish وتكوينه والعمليات اليومية.

## روابط سريعة

- **هل آنجا شيء باطّل الآن؟** ابدأ بـ[دليل عیب‌یابی](/fa-IR/support/troubleshooting/)
- **هل تحتاج به البحث عن خطأ؟** مراجعه کنید [مرجع خطاها](/fa-IR/support/troubleshooting/error-reference)
- **هل تريد الإبخیرغ عن خلل؟** اقرأ [نحوه تقديم بخیرغ جيد](/fa-IR/support/guides/filing-issues) یاخیرً
- **هل تقوم بالترقية یا الترحيل؟** تحقق از [پایگاه دانش](#قانون-البارفة)

## موارد الخدمة الذاتية

### عیب‌یابی

راهنماها خطوة بخطوة لتشخيص المشكخیرت الشائعة وإصخیرحها، ازظمة حسب المجال:

| المجال | يغطي |
|------|--------|
| [نصب](/fa-IR/support/troubleshooting/installation) | فشل نصب، أخطاء الأذونات، راه‌اندازی حسب الازصة |
| [الخدمة الخلدرة](/fa-IR/support/troubleshooting/daemon) | مشاهر شروع/الإيقاف، إدارة الخدمة، تعارضات الازافذ |
| [پیکربندی](/fa-IR/support/troubleshooting/configuration) | تحليل YAML، أخطاء التحقق، ممراجعه کنید رمزها |
| [کانال‌ها](/fa-IR/support/troubleshooting/channels) | Telegram، Slack، Discord، WhatsApp، Signal، Email، WebChat |
| [ارائه‌دهندگان LLM](/fa-IR/support/troubleshooting/providers) | أخطاء مفاتيح API، مدل غير موجود، فشل البث، التبديل الخودکار |
| [یکپارچه‌سازی‌ها](/fa-IR/support/troubleshooting/integrations) | Google، GitHub، Notion، CalDAV، خوادم MCP |
| [اتوماسیون مرورگر](/fa-IR/support/troubleshooting/browser) | اكتشاف Chrome، فشل التشغيل، Flatpak، التنقل |
| [اازیت وطبقه‌بندی](/fa-IR/support/troubleshooting/security) | تصعيد Taint، حظر نوشتن به پایین، SSRF، رفض سیاست‌ها |
| [رمزها وبيانات اخیرعتماد](/fa-IR/support/troubleshooting/secrets) | واجهات سلسلة المفاتيح، أخطاء الأذونات، مخزن الملفات المشفر |
| [مرجع خطاها](/fa-IR/support/troubleshooting/error-reference) | فهرس قابل للبحث لتمام پیام‌ها الخطأ |

## هل ما زلت عالقاً؟

إذا لم تحل الوثائق أعخیره مشهرتك:

1. **ابحث در البخیرغات الموجودة** روی [GitHub Issues](https://github.com/greghavens/triggerfish/issues) لبارفة ما إذا كان شخص ما قد أبلغ عنها در واقع
2. **اسأل المجتبا** در [GitHub Discussions](https://github.com/greghavens/triggerfish/discussions)
3. **قدّم بخیرغاً جديداً** باتباع [دليل تقديم البخیرغات](/fa-IR/support/guides/filing-issues)
