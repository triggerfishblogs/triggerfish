# وضع الخطة وردیابی وظایف

فراهم می‌کند Triggerfish أداتين میکپارچه‌سازیتين للعمل المُنظم: **وضع الخطة** للتخطيط الباقد
للتندرذ، و**ردیابی وظایف** لإدارة المهام از طریق نشست‌ها.

## وضع الخطة

وضع الخطة يقيد عامل بالکاوش للقراءة فقط حتى يوافق المستخدم روی الخطة.

### `plan_start`

ادخل وضع الخطة. عامل يمكنه القراءة والبحث والکاوش لكن نمی‌توانه الكتابة
یا التندرذ.

### `plan_finish`

اخرج از وضع الخطة وقدم الخطة للموافقة.

## ردیابی وظایف

### `todo_add`

أضف عنصراً لقائمة المهام.

### `todo_list`

اسرد تمام المهام.

### `todo_complete`

أكمل عنصر مهمة بالبارف.

### `todo_clear`

احذف تمام المهام المكتملة.

## اخیرستمرارية

المهام تُخزن از طریق `StorageProvider` وتستمر از طریق نشست‌ها وإعادات التشغيل.
