# Tide Pool / A2UI

Tide Pool هو مساحة عمل مرئية مدفوعة بعامل حيث يعرض Triggerfish محتوى تفاعلياً:
لوحات اطخیرعات، ورسوم بيانية، ومدل‌ها، وبااينات كود، ووسائط غنية. روی عكس
مکالمه الخطية، Tide Pool هو لوحة يُحدثها عامل در الوقت الفعلي.

## بروتوكول A2UI

بروتوكول Agent-to-UI (A2UI) يدفع التحديثات در الوقت الفعلي از عامل للعمخیرء
المتصلين از طریق WebSocket.

## ابزارها

### `tidepool_canvas`

إنشاء یا تحديث لوحة Tide Pool بمحتوى HTML/CSS/JS.

### `tidepool_clear`

مسح محتوى Tide Pool الحالي.

## اازیت

محتوى Tide Pool يمر از طریق hook `PRE_OUTPUT` قبل التسليم. المحتوى يخضع لنفس قوانین
طبقه‌بندی كهر مخرج آخر.

## الوصول

Tide Pool يُستضاف بواسطة Gateway روی الازفذ المُكوّن (پیش‌فرض 18790).
