# جستجوی وب والجلب

يازح Triggerfish عاملك وصوخیرً للإنترنت از خخیرل أداتين: `web_search` للبحث عن
اخیرطخیرعات و `web_fetch` لقراءة صفحات الويب. بااً يتيحان للعامل البحث در
المواضيع وممراجعه کنیدة الوثائق والتحقق از الحقائق.

## ابزارها

### `web_search`

جستجوی وب باستخدام ارائه‌دهنده البحث المُكوّن.

| الباامل       | النوع  | الزامی | الوصف                              |
| ------------- | ------ | ----- | ---------------------------------- |
| `query`       | string | بله   | استعخیرم البحث                       |
| `max_results` | number | خیر    | أقصى عدد نتائج (پیش‌فرض: 10)       |

### `web_fetch`

جلب وقراءة محتوى صفحة ويب.

| الباامل | النوع  | الزامی | الوصف                              |
| ------- | ------ | ----- | ---------------------------------- |
| `url`   | string | بله   | عنوان URL للجلب                    |
| `mode`  | string | خیر    | `readability` (پیش‌فرض) یا `raw`   |

## ارائه‌دهنده البحث

در حال حاضر پشتیبانی می‌کند Brave Search. كوّن در `triggerfish.yaml`:

```yaml
web:
  search:
    provider: brave
```

## اازیت

### ازع SSRF

تمام درخواستات HTTP الصادرة تحل DNS یاخیرً وتفحص عنوان IP مقابل قائمة حظر مُرمزة
از النطاقات الخاصة. این يازع الوصول لخدمات داخلية از طریق عناوين URL مُصاغة.

### طبقه‌بندی النطاقات

می‌توانید طبقه‌بندی النطاقات لتندرذ taint عند الوصول لمواقع محددة:

```yaml
web:
  domains:
    classifications:
      - pattern: "*.internal.corp"
        classification: CONFIDENTIAL
```

### قوائم السماح والرفض

```yaml
web:
  domains:
    denylist:
      - "*.malware-site.com"
    allowlist: []
```
