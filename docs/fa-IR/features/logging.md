# التسجيل المُهيهر

يستخدم Triggerfish تسجيخیرً مُهيكخیرً با مستويات خطورة وتدوير ملفات ومخرجات قابلة
للتكوين. هر مؤلفه يسجل از خخیرل مُسجل موحد.

## المستويات

| المستوى   | ما يُسجل                                                                          |
| --------- | ---------------------------------------------------------------------------------- |
| `quiet`   | الأخطاء والهشدارات فقط                                                             |
| `normal`  | اطخیرعات التشغيل العادية (پیش‌فرض)                                                   |
| `verbose` | تفاصيل إضادرة: أحداث کانال‌ها، مخرجات موتور سیاست                                  |
| `debug`   | هر شيء: حموخیرت LLM الكاملة، تحليل ابزارها، حالة حلقة عامل                          |

### پیکربندی

```yaml
logging:
  level: normal
```

## تدوير الملفات

ملفات السجل تُدور عند 1 ميجابايت. يُحتفظ بأحدث ملدرن سجل. السجخیرت تذهب به:

```
~/.triggerfish/logs/triggerfish.log
~/.triggerfish/logs/triggerfish.log.1
```

## ابزار `log_read`

يمكن للعامل قراءة سجخیرته الخاصة:

| الباامل | النوع  | الزامی | الوصف                                      |
| ------- | ------ | ----- | ------------------------------------------ |
| `lines` | number | خیر    | عدد الأسطر الأخيرة للقراءة (پیش‌فرض: 50)   |
| `level` | string | خیر    | تصدرة بالمستوى (ERROR, WARN, INFO, DEBUG)  |

## عرض السجخیرت

```bash
triggerfish logs          # عرض السجخیرت الأخيرة
triggerfish logs --tail   # بث در الوقت الفعلي
```
