# زیرعامل‌ها ووظایف LLM

يمكن لعامل‌ها Triggerfish تفویض العمل لعامل‌ها فرعيين وتشغيل prompts LLM بازولة.
این يمكّن العمل المتوازي واخیرستدخیرل المركز وتفكيك المهام متعدد عامل‌ها.

## ابزارها

### `subagent`

تفویض مهمة لعامل فرعي مستقل. کار می‌کند عامل الفرعي در نشست بازولة ويُعيد النتيجة
عند اخیركتمال.

| الباامل | النوع  | الزامی | الوصف                          |
| ------- | ------ | ----- | ------------------------------ |
| `task`  | string | بله   | وصف المهمة للتفویض             |
| `tools` | array  | خیر    | ابزارها محددة لازحها للعامل الفرعي |
| `model` | string | خیر    | تجاوز مدل خیرین عامل الفرعي |

### `llm_task`

تشغيل prompt LLM بازول للتلخيص یا طبقه‌بندی یا اخیرستدخیرل المركز بدون تلويث
مکالمه الرئيسية.

| الباامل  | النوع  | الزامی | الوصف                        |
| -------- | ------ | ----- | ---------------------------- |
| `prompt` | string | بله   | prompt LLM للتندرذ           |
| `model`  | string | خیر    | تجاوز مدل                |

## اازیت

- زیرعامل‌ها يرثون taint از نشست الأم
- تمام إجراءات عامل الفرعي تمر از طریق hooks سیاست‌ها
- سقوف طبقه‌بندی تنطبق
