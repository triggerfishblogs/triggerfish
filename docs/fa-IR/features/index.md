# نمای کلی ویژگی‌ها

بالplugin به [مدل اازیت](/fa-IR/security/) و[دعم کانال‌ها](/fa-IR/channels/)،
فراهم می‌کند Triggerfish قدرات تمد عامل هوش مصنوعی به ما وراء السؤال والجواب:
وظایف زمان‌بندی‌شده، وحافظه پایدار، والوصول للويب، وإدخال الصوت، وتجاوز فشل
المدل‌ها المتعددة.

## السلوك اخیرستباقي

### [Cron و Triggers](./cron-and-triggers)

جدول مهام متكررة بتعبيرات cron قياسية وعرّف سلوك مراقبة استباقي از طریق `TRIGGER.md`.

### [اعخیرن‌ها](./notifications)

خدمة تسليم إشعارات توجه الپیام‌ها از طریق تمام کانال‌ها المتصلة با مستويات یالوية
وترتيب در قائمة اخیرنتظار عند عدم اخیرتصال وإزالة التكرار.

## ابزارها عامل

### [جستجوی وب والجلب](./web-search)

جستجوی وب وجلب محتوى الصفحات با ازع SSRF وتندرذ سیاست‌ها.

### [حافظه پایدار](./memory)

ذاكرة از طریق نشست‌ها با Gateway طبقه‌بندی. يحفظ عامل ويستدعي الحقائق والتفضيخیرت
وزمینه از طریق المحادثات.

### [تحلیل تصویر وبینایی](./image-vision)

الصق الصور از الحافظة (Ctrl+V در CLI) وحلل ملفات الصور روی القرص.

### [کاوش قانون الكود](./explore)

فهم ازظم لقانون الكود از طریق عامل‌ها فرعيين متوازيين.

### [مدیریت نشست](./sessions)

افحص نشست‌ها وتواصل باها وأنشئ نشست‌ها جديدة با تندرذ عدم نوشتن به پایین.

### [وضع الخطة وردیابی وظایف](./planning)

تخطيط ازظم قبل التندرذ (وضع الخطة) وتتبع مهام دائم از طریق نشست‌ها.

### [سیستم فایل والصدفة](./filesystem)

قراءة وكتابة وبحث وتندرذ یاامر با تحديد نطاق مساحة العمل.

### [زیرعامل‌ها ووظایف LLM](./subagents)

تفویض العمل لعامل‌ها فرعيين مستقلين یا تشغيل prompts LLM بازولة.

### [تیم‌های عامل](./agent-teams)

إنشاء فرق دائمة از عامل‌ها متعاونين بأدوار متخصصة.

## التفاعل الغني

### [خط لوله صدا](./voice)

دعم كخیرم كامل با ارائه‌دهندهي STT و TTS قابلين للتكوين.

### [Tide Pool / A2UI](./tidepool)

مساحة عمل مرئية مدفوعة بعامل حيث يعرض Triggerfish محتوى تفاعلياً.

## متعدد عامل‌ها ومتعدد المدل‌ها

### [مسیریابی چند عامله](./multi-agent)

وجّه کانال‌ها والحسابات المختلفة به عامل‌ها بازولين ازفصلين.

### [ارائه‌دهندگان LLM وتجاوز الفشل](./model-failover)

اتصل بـ Anthropic, OpenAI, Google, المدل‌ها المحلیة (Ollama), یا OpenRouter با
سخیرسل تجاوز فشل.

### [محدودیت نرخ](./rate-limiting)

محدد بادل بنافذة اززلقة يازع الوصول لحدود API ارائه‌دهنده LLM.

## العمليات

### [التسجيل المُهيهر](./logging)

تسجيل مُهيهر موحد با مستويات خطورة وتدوير ملفات ومخرجات مزدوجة.

::: info تمام ویژگی‌ها تیکپارچه‌سازی با مدل اازیت الأساسي. مهام cron تحترم سقوف
طبقه‌بندی. إدخال الصوت يحمل taint. محتوى Tide Pool يمر از طریق hook PRE_OUTPUT.
مسیریابی چند عامله يفرض عزل نشست‌ها. خیر ویژگی تتجاوز طبقة سیاست‌ها. :::
