# اعخیرن‌ها

NotificationService هو تجريد Triggerfish از الدرجة الیالى لتسليم اعخیرن‌ها
لمالك عامل از طریق تمام کانال‌ها المتصلة.

## مستويات الیالوية

| الیالوية   | السلوك                                                            |
| ---------- | ----------------------------------------------------------------- |
| `CRITICAL` | تجاوز ساعات الهدوء، تسليم لتمام کانال‌ها فوراً                     |
| `HIGH`     | تسليم للکانال المفضلة فوراً، ترتيب إذا غير متصل                    |
| `NORMAL`   | تسليم للنشست النشطة، یا ترتيب للنشست التالية                      |
| `LOW`      | ترتيب، تسليم روی دفعات أثناء نشست‌ها النشطة                       |

## الفئات

| الفئة      | مثال‌ها                                                             |
| ---------- | ----------------------------------------------------------------- |
| `security` | انتهاكات سیاست‌ها، تنبيهات التهديد                                 |
| `approval` | درخواستات الموافقة روی مهارت‌ها                                        |
| `system`   | فشل cron، هشدارات صحة النظام                                      |
| `info`     | أحداث webhook، تحديثات The Reef                                   |

## التفضيخیرت

```yaml
notifications:
  preferred_channel: telegram
  quiet_hours:
    start: "22:00"
    end: "07:00"
    timezone: "America/Chicago"
  overrides:
    security: all_channels
    approval: preferred_channel
    info: active_session
```

## اخیرستمرارية

اعخیرن‌ها تُخزن از طریق `StorageProvider` (مساحة اخیرسم: `notifications:`) وتنجو از
إعادات التشغيل.
