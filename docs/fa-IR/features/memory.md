# حافظه پایدار

عامل‌ها Triggerfish لديهم ذاكرة دائمة از طریق نشست‌ها. يمكن للعامل حفظ الحقائق
والتفضيخیرت وزمینه الذي ينجو از طریق المحادثات وإعادات التشغيل. حافظه محمية
بطبقه‌بندی -- خیر يستطيع عامل قراءة ذكريات فوق مستوى taint جلسته الحالية.

## ابزارها

### `memory_save`

حفظ حقيقة یا تفضيل یا زمینه در حافظه طويلة الأمد.

| الباامل    | النوع  | الزامی | الوصف                              |
| ---------- | ------ | ----- | ---------------------------------- |
| `content`  | string | بله   | چه يُحفظ                         |
| `category` | string | خیر    | فئة اختیاریة (preference, fact)    |

### `memory_search`

البحث در حافظه المحفوظة باستخدام بحث نص كامل.

| الباامل | النوع  | الزامی | الوصف            |
| ------- | ------ | ----- | ---------------- |
| `query` | string | بله   | نص البحث          |

### `memory_list`

سرد تمام إدخاخیرت حافظه المحفوظة.

### `memory_delete`

حذف إدخال ذاكرة بالبارف.

## Gateway طبقه‌بندی

- **الكتابة**: تُجبر إدخاخیرت حافظه روی مستوى taint نشست الحالية
- **القراءة**: تُفلتر استعخیرمات حافظه بـ `canFlowTo`

::: warning SECURITY خیر يستطيع LLM اختيار طبقه‌بندی حافظه. انجام می‌شود فرض طبقه‌بندی روی
مستوى taint نشست در hook `PRE_TOOL_CALL`. :::

## ذخیره‌سازی

حافظه تُخزن از طریق `StorageProvider` با بحث نص كامل SQLite FTS5 للبحث الفعال.
