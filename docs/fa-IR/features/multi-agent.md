# مسیریابی چند عامله

پشتیبانی می‌کند Triggerfish توجيه کانال‌ها والحسابات وجهات اخیرتصال المختلفة به عامل‌ها بازولين
ازفصلين، هر ازها بمساحة عمله ونشست‌هاه وشخصيته وسقف طبقه‌بندیه.

## چگونه کار می‌کند

وجّه عامل‌ها مختلدرن لقنوات مختلفة:

- Slack العمل يذهب لعامل واحد
- WhatsApp الشخصي يذهب لعامل آخر
- هر عامل لديه SPINE.md خاص ومساحة عمل ومهارات وتتبع taint

## تكوين عامل

هر عامل لديه:

- `SPINE.md` خاص در `~/.triggerfish/workspace/<agent-id>/`
- `TRIGGER.md` اختیاری خاص
- نشست‌ها بازولة بتتبع taint مستقل
- سقف طبقه‌بندی يحد از البيانات التي يمكنه الوصول إليها

## عزل اازیت

- عامل‌ها نمی‌توانهم الوصول لنشست‌ها یا مساحات عمل بعضهم
- اخیرتصال بین عامل‌ها يخضع لسخیرسل تفویض وعدم نوشتن به پایین
- هر عامل لديه سقف طبقه‌بندی مستقل
