# مدیریت نشست

يمكن للعامل فحص نشست‌ها والتواصل باها وإنشاء نشست‌ها جديدة. تمكّن این ابزارها
سير العمل از طریق نشست‌ها وتفویض المهام الخلدرة والمراسلة از طریق کانال‌ها -- هر آن
تحت تندرذ عدم نوشتن به پایین.

## ابزارها

### `sessions_list`

سرد نشست‌ها النشطة با مرشحات اختیاریة. خیر تغيير در taint.

### `sessions_history`

استرجاع نص نشست. Taint يرث از نشست المُشار إليها.

### `sessions_send`

إرسال پیام لنشست أخرى. يخضع لفحص عدم نوشتن به پایین.

### `sessions_spawn`

إنشاء نشست مهمة خلدرة. تبدأ بـ taint `PUBLIC`.

### `session_status`

فحص حالة نشست الحالية: مدل، عدد الرموز، التهرفة، مستوى taint.
