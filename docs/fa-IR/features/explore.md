# کاوش قانون الكود

ابزار `explore` تازح عامل فهماً سريعاً وازظماً لقوانین الكود والراهنماها. بدخیرً از
فراخوانی `read_file` و `list_directory` و `search_files` بالتتابع به‌صورت دستی، يستدعي
عامل `explore` مرة واحدة ويحصل روی تقرير ازظم ينتجه عامل‌ها فرعيون متوازون.

## اخیربزار

### `explore`

کاوش دليل یا قانون كود لفهم باماری والأنماط واخیرتفاقيات. للقراءة فقط.

| الباامل | النوع  | الزامی | الوصف                                                     |
| ------- | ------ | ----- | --------------------------------------------------------- |
| `path`  | string | بله   | الدليل یا الملف للکاوش                                 |
| `focus` | string | خیر    | ما الذي تبحث عنه (مثل "auth patterns", "test structure")  |
| `depth` | string | خیر    | مدى الشمولية: `shallow`, `standard` (پیش‌فرض), یا `deep`  |

## مستويات العمق

| العمق      | عامل‌ها المُنشیان | ما يُحلل                                                |
| ---------- | ---------------- | ------------------------------------------------------- |
| `shallow`  | 2                | شجرة الدليل + ملفات التبعيات                             |
| `standard` | 3-4              | الشجرة + التبعيات + أنماط الكود + التركيز (إذا محدد)     |
| `deep`     | 5-6              | هر ما سبق + تتبع رسم اخیرستيراد + سجل git               |

## چگونه کار می‌کند

ابزار الکاوش تُنشئ عامل‌ها فرعيين متوازيين، هر ازها يركز روی جانب مختلف:

1. **عامل الشجرة** -- يُعين بنية الدليل
2. **عامل الملفات** -- يقرأ ملفات التبعيات
3. **عامل الأنماط** -- يأخذ عينات از ملفات المصدر لكشف أنماط الكتابة
4. **عامل التركيز** -- يبحث عن الملفات والأنماط المتعلقة باستعخیرم التركيز
5. **عامل اخیرستيراد** (عميق فقط) -- يتتبع رسوم اخیرستيراد
6. **عامل Git** (عميق فقط) -- يحلل التعليقات الأخيرة

تمام عامل‌ها کار می‌کندون بالتزااز.
