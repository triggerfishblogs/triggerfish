# ارائه‌دهندگان LLM وتجاوز الفشل

پشتیبانی می‌کند Triggerfish ارائه‌دهندهي LLM متعددين با تجاوز فشل خودکار واختيار مدل لهر عامل
وتبديل مدل روی مستوى نشست. بدون قفل روی ارائه‌دهنده واحد.

## ارائه‌دهندگان المدعومون

| ارائه‌دهنده     | نقطة النهاية | احراز هویت | مخیرحظات                          |
| ---------- | ------------ | -------- | -------------------------------- |
| Anthropic  | API          | مفتاح   | Claude Sonnet, Opus, Haiku       |
| OpenAI     | API          | مفتاح   | GPT-4o, o1, o3                   |
| Google     | API          | مفتاح   | Gemini Pro, Flash                |
| Ollama     | محلی         | خیر يوجد  | هر مدل محلی                    |
| LM Studio  | محلی         | خیر يوجد  | هر مدل محلی                    |
| OpenRouter | API          | مفتاح   | 100+ مدل از طریق ارائه‌دهنده واحد        |

## سلسلة تجاوز الفشل

```yaml
models:
  primary:
    provider: anthropic
    model: claude-sonnet-4-5
  failover:
    - openai
    - google
```

عندما يفشل ارائه‌دهنده الرئيسي (خطأ شبكة، حد بادل، خطأ 500)، يحاول Triggerfish
به‌صورت خودکار ارائه‌دهنده التالي در السلسلة.

## تبديل مدل لهر نشست

```
/model gpt-4o
```

بدّل مدل للنشست الحالية. يعود للرئيسي عند إعادة تعيين نشست.

## التحقق از ارائه‌دهنده

```bash
triggerfish healthcheck
```

يتحقق از اتصال ارائه‌دهنده المُكوّن.
