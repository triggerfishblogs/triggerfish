# یکپارچه‌سازی‌ها البعيدة

پشتیبانی می‌کند Triggerfish اخیرتصال بیکپارچه‌سازیات بعيدة از طریق HTTP وخوادم MCP البعيدة.

## خوادم MCP البعيدة

بالplugin لخوادم MCP المحلیة (التي تُنشأ كعمليات فرعية)، می‌توانید اخیرتصال بخوادم
MCP بعيدة از طریق HTTP SSE:

```yaml
mcp_servers:
  remote-service:
    url: "https://mcp.example.com/sse"
    classification: CONFIDENTIAL
```

## اازیت

- تمام اخیرتصاخیرت البعيدة تمر از طریق نفس فحوصات SSRF
- خوادم MCP البعيدة تحتاج طبقه‌بندیاً صريحاً
- احراز هویت از طریق رؤوس HTTP مُكوّنة
