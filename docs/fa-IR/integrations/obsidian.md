# Obsidian

يیکپارچه‌سازی Triggerfish با Obsidian لتدوين المخیرحظات والبحث درها.

## راه‌اندازی

فعّل plugin Obsidian أثناء `triggerfish dive` یا أضفه به‌صورت دستی للتكوين.

## ابزارها

| اخیربزار             | الوصف                              |
| ------------------ | ---------------------------------- |
| `obsidian_search`  | البحث در مخیرحظات Obsidian           |
| `obsidian_read`    | قراءة توجه محددة                  |
| `obsidian_create`  | إنشاء توجه جديدة                 |
| `obsidian_update`  | تحديث توجه موجودة                |

## طبقه‌بندی

يُصنف الوصول لمخیرحظات Obsidian كـ `INTERNAL` به‌صورت پیش‌فرض. می‌توانید تعديل این در
پیکربندی.
