# ساخت مهارت‌ها

مهارت‌ها توسع قدرات عاملك ببارفة ومهام متخصصة. ملف `SKILL.md` در هر مجلد
يُعلم Triggerfish أن این المجلد يحتوي روی مهارت.

## بنية المهارت

```
my-skill/
├── SKILL.md          # تعريف المهارت (الزامی)
├── templates/        # قوالب اختیاریة
└── examples/         # مثال‌ها اختیاریة
```

## SKILL.md

ملف SKILL.md يحدد:

- **اخیرسم والوصف**: ما تفعله المهارت
- **المُشغخیرت**: چه زمانی باید تفعيل المهارت
- **التعليمات**: چگونه باید روی عامل استخدام المهارت
- **ابزارها**: هر ابزارها خاصة تحتاجها المهارت

## أنواع مهارت‌ها

| النوع     | الموقع                                        | الوصف                        |
| --------- | --------------------------------------------- | ---------------------------- |
| مُدمجة    | `src/skills/`                                 | تُشحن با Triggerfish         |
| مُدارة    | `~/.triggerfish/skills/`                       | مُثبتة از The Reef           |
| مساحة عمل | `~/.triggerfish/workspace/<agent>/skills/`     | مُنشأة بواسطة عامل         |

## النشر

```bash
triggerfish skill publish
```

ينشر مهارتك در The Reef للآخرين خیرستخدامها.
