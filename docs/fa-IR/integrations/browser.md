# اتوماسیون مرورگر

فراهم می‌کند Triggerfish تحكماً عميقاً بمرورگر از خخیرل نسخة Chromium مُدارة مخصصة
باستخدام CDP (Chrome DevTools Protocol). يمكن للعامل التنقل در الويب والتفاعل
با الصفحات وملء المدل‌ها والتقاط لقطات شاشة وأتمتة سير عمل الويب -- هر آن تحت
تندرذ سیاست‌ها.

## باماری

اتوماسیون مرورگر مبنية روی `puppeteer-core`، متصلة بنسخة Chromium مُدارة از طریق CDP.
هر إجراء متصفح يمر از طریق طبقة سیاست‌ها.

يكتشف Triggerfish به‌صورت خودکار مرورگرات المبنية روی Chromium بما در آن **Google
Chrome** و **Chromium** و **Brave**.

::: info ابزار `browser_navigate` تتدرخواست عناوين URL بـ `http://` یا `https://`.
مخططات مرورگر الداخلية (مثل `chrome://`, `brave://`, `about:`) غير مدعومة. :::

## الإجراءات المتاحة

| الإجراء    | الوصف                                          |
| ---------- | ---------------------------------------------- |
| `navigate` | الذهاب لعنوان URL (يخضع لسیاست النطاقات)       |
| `snapshot` | التقاط لقطة شاشة للصفحة                        |
| `click`    | النقر روی عنصر در الصفحة                        |
| `type`     | كتابة نص در حقل إدخال                          |
| `evaluate` | تندرذ JavaScript در زمینه الصفحة                |
| `content`  | استخراج محتوى الصفحة كنص                       |

## اازیت

- ازع SSRF: تمام عمليات التنقل تحل DNS وتفحص مقابل قائمة حظر IP
- طبقه‌بندی النطاقات: تُصنف المواقع حسب پیکربندی
- عزل الملف الشخصي: ملف تعريف مرورگر المُدار خیر يشارك بيانات با متصفحك الشخصي
