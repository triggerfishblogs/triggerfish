# شروع کار

Triggerfish یک پلتفرم عامل هوش مصنوعی امن و چندکاناله با اعمال قطعی سیاست‌ها
در زیر لایه LLM است. به هر پلتفرم پیام‌رسانی که قبلاً استفاده می‌کنید متصل
می‌شود — Telegram، Slack، Discord، WhatsApp، ایمیل، WebChat و CLI — در حالی
که قوانین طبقه‌بندی داده‌ها را اعمال می‌کند که هوش مصنوعی نمی‌تواند آن‌ها را دور
بزند، لغو کند یا بر آن‌ها تأثیر بگذارد.

چه فردی باشید که یک دستیار هوش مصنوعی شخصی قدرتمند با تضمین‌های امنیتی واقعی
می‌خواهد، یا سازمانی که به اعمال سیاست‌های قابل بازرسی در تعاملات کارمند-عامل
نیاز دارد، Triggerfish هم وسعت یکپارچه‌سازی و هم کنترل‌های لازم برای استفاده
مطمئن از هوش مصنوعی را ارائه می‌دهد.

Triggerfish متن‌باز (مجوز Apache 2.0) است، به‌صورت محلی روی دستگاه شما اجرا
می‌شود و از ارائه‌دهندگان متعدد LLM از جمله Anthropic Claude، OpenAI، Google
Gemini، مدل‌های محلی از طریق Ollama و OpenRouter پشتیبانی می‌کند.
