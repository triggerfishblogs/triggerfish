# مرجع

فراهم می‌کند این القسم مواد مرجعية تقنية لتكوين Triggerfish وتوسيعه.

## [شمای پیکربندی](./config-yaml)

مرجع كامل لـ `triggerfish.yaml` -- هر قسم وهر مفتاح، با مثال مُعلق. يغطي
المدل‌ها وکانال‌ها وطبقه‌بندی وسیاست‌ها وخوادم MCP والمجدول واعخیرن‌ها.

## [رابط‌های کلیدی](./interfaces)

مرجع واجهات TypeScript للمطورين الذين يبنون یکپارچه‌سازیات یا محوخیرت قنوات مخصصة یا
ارائه‌دهندهي LLM یا واجهات ذخیره‌سازی خلدرة.

## [واژه‌نامه](./glossary)

تعريفات لهر مصطلح خاص بـ Triggerfish، از Buoy به Write-Down.
