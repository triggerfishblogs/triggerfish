# واژه‌نامه

تعريفات للمصطلحات الخاصة بـ Triggerfish.

| المصطلح | التعريف |
|---------|---------|
| **A2UI** | بروتوكول Agent-to-UI. يدفع التحديثات در الوقت الفعلي از عامل لواجهات المستخدم المتصلة. |
| **Buoy** | تطبيق مرافق فراهم می‌کند قدرات الجهاز (الكاميرا، الموقع، إلخ.) |
| **Classification** | مستوى حساسية البيانات (RESTRICTED > CONFIDENTIAL > INTERNAL > PUBLIC) |
| **Dive** | باالج راه‌اندازی الیال (`triggerfish dive`). يُنشئ SPINE.md و triggerfish.yaml |
| **Gateway** | مستوى التحكم المركزي. خدمة محلیة طويلة التشغيل تدير نشست‌ها وکانال‌ها وابزارها |
| **Hook** | نقطة تندرذ قطعیة در جریان داده |
| **MCP** | Model Context Protocol. بروتوكول لتوصيل ابزارها وبيانات خارجية |
| **No Write-Down** | قانون تازع جریان داده لمستوى طبقه‌بندی أقل |
| **Patrol** | أمر فحص صحة تشخيصي (`triggerfish patrol`) |
| **Plugin** | كود خارجي کار می‌کند داخل عزل مزدوج (Deno + WASM) |
| **Reef, The** | سوق مهارت‌ها خیركتشاف وتثبيت ونشر مهارت‌ها |
| **Ripple** | مؤشرات الكتابة وحالة اخیرتصال از طریق کانال‌ها |
| **SDK** | Plugin SDK. واجهة للإضافات الخارجية |
| **Session** | الوحدة الأساسية لحالة مکالمه با تتبع taint مستقل |
| **Skill** | مجلد يحتوي SKILL.md يازح عامل قدرات جديدة |
| **SPINE.md** | ملف هویت ومهمة عامل (أساس prompt النظام) |
| **SSRF** | تزوير درخواستات از جانب الخادم. يُازع بقائمة حظر IP مُرمزة |
| **Taint** | طبقه‌بندی روی مستوى نشست يتصاعد عند الوصول لبيانات مصنفة. نمی‌توانه اخیرنخفاض |
| **Tide Pool** | مساحة عمل مرئية مُعرضة از طریق بروتوكول A2UI |
| **TRIGGER.md** | تعريف السلوك اخیرستباقي للعامل |
| **Trigger** | إيقاظ عامل دوري للسلوك اخیرستباقي المستقل |
| **Write-Down** | تدفق محظور للبيانات لمستوى طبقه‌بندی أقل |
