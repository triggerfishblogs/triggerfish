# 支援中心

取得 Triggerfish 安裝、設定及日常操作的協助。

## 快速連結

- **遇到問題了嗎？** 請從[疑難排解指南](/zh-TW/support/troubleshooting/)開始
- **需要查詢錯誤訊息？** 請參閱[錯誤訊息參考](/zh-TW/support/troubleshooting/error-reference)
- **想要回報 Bug？** 請先閱讀[如何提交好的 Issue](/zh-TW/support/guides/filing-issues)
- **正在升級或遷移？** 請查閱[知識庫](#知識庫)

## 仍然無法解決？

如果以上文件未能解決您的問題：

1. **搜尋現有 Issue**：在 [GitHub Issues](https://github.com/greghavens/triggerfish/issues) 上查看是否已有人回報
2. **詢問社群**：在 [GitHub Discussions](https://github.com/greghavens/triggerfish/discussions) 上發問
3. **提交新 Issue**：請依照[提交 Issue 指南](/zh-TW/support/guides/filing-issues)
