# 參考

本節提供配置和擴展 Triggerfish 的技術參考資料。

## [設定結構](./config-yaml)

`triggerfish.yaml` 的完整參考——每個區段、每個鍵，帶有註解範例。涵蓋模型、通道、分類、策略、MCP 伺服器、排程器和通知。

## [關鍵介面](./interfaces)

為建構整合、自訂通道 adapter、LLM 提供者或儲存後端的開發者提供的 TypeScript 介面參考。包括 `StorageProvider`、`ChannelAdapter`、`LlmProvider`、`NotificationService`、hook 類型、工作階段狀態和 `Result<T, E>` 模式。

## [詞彙表](./glossary)

每個 Triggerfish 專有術語的定義，從 Buoy 到 Write-Down。
