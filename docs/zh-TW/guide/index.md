# 入門指南

Triggerfish 是一個安全的多頻道 AI 代理平台，在 LLM 層之下提供確定性策略執行。
它連接到您已經使用的每個訊息平台——Telegram、Slack、Discord、WhatsApp、Email、
WebChat 和 CLI——同時執行 AI 無法繞過、覆寫或影響的資料分類規則。

無論您是想要一個具有真正安全保證的強大個人 AI 助手的個人使用者，還是需要在
員工與代理互動中進行可稽核策略執行的組織，Triggerfish 都能提供整合廣度和控制，
讓您有信心地使用 AI。

Triggerfish 是開放原始碼的（Apache 2.0 授權），在您的機器上本機執行，並支援
多個 LLM 提供者，包括 Anthropic Claude、OpenAI、Google Gemini、透過 Ollama
使用的本地模型和 OpenRouter。
