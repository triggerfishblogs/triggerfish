# Support Centre

Get help with Triggerfish installation, configuration, and day-to-day operations.

## Quick Links

- **Something broken right now?** Start with the [Troubleshooting Guide](/en-GB/support/troubleshooting/)
- **Need to look up an error?** See the [Error Reference](/en-GB/support/troubleshooting/error-reference)
- **Want to file a bug?** Read [How to File a Good Issue](/en-GB/support/guides/filing-issues) first
- **Upgrading or migrating?** Check the [Knowledge Base](#knowledge-base)

## Still Stuck?

If the docs above did not solve your problem:

1. **Search existing issues** on [GitHub Issues](https://github.com/greghavens/triggerfish/issues) to see if someone already reported it
2. **Ask the community** in [GitHub Discussions](https://github.com/greghavens/triggerfish/discussions)
3. **File a new issue** following the [issue filing guide](/en-GB/support/guides/filing-issues)
