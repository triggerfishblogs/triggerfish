# Rujukan

Bahagian ini menyediakan bahan rujukan teknikal untuk mengkonfigurasi dan memanjangkan Triggerfish.

## [Skema Config](./config-yaml)

Rujukan penuh untuk `triggerfish.yaml` -- setiap bahagian, setiap kunci, dengan contoh beranotasi. Merangkumi model, saluran, pengkelasan, dasar, pelayan MCP, penjadual, dan pemberitahuan.

## [Antara Muka Utama](./interfaces)

Rujukan antara muka TypeScript untuk pembangun yang membina integrasi, penyesuai saluran tersuai, pembekal LLM, atau backend storan. Termasuk `StorageProvider`, `ChannelAdapter`, `LlmProvider`, `NotificationService`, jenis hook, keadaan sesi, dan corak `Result<T, E>`.

## [Glosari](./glossary)

Definisi untuk setiap istilah khusus Triggerfish, dari Buoy hingga Write-Down.
