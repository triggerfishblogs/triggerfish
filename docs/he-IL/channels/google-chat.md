# Google Chat

<ComingSoon />

חברו את סוכן ה-Triggerfish שלכם ל-Google Chat כדי שצוותים המשתמשים
ב-Google Workspace יוכלו לתקשר איתו ישירות מממשק הצ'אט שלהם. המתאם
ישתמש ב-Google Chat API עם אישורי חשבון שירות או OAuth.

## תכונות מתוכננות

- תמיכה בהודעות ישירות ומרחבים (חדרים)
- אימות בעלים דרך ספריית Google Workspace
- מחווני הקלדה
- חלוקת הודעות לתגובות ארוכות
- אכיפת סיווג עקבית עם ערוצים אחרים

## תצורה (מתוכננת)

```yaml
channels:
  google-chat:
    classification: INTERNAL
```

ראו [Google Workspace](/he-IL/integrations/google-workspace) לאינטגרציית
Google הקיימת המכסה Gmail, Calendar, Tasks, Drive ו-Sheets.
