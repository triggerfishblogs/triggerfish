# גישה מרחוק

<ComingSoon />

גישה מרחוק מאפשרת לכם להגיע למופע Triggerfish שלכם מחוץ לרשת המקומית.
זה נדרש למסירת webhook משירותים חיצוניים (GitHub, Stripe, וכו') ולגישה
לסוכן ממכשירים ניידים כאשר אתם מחוץ לבית.

## אפשרויות מתוכננות

| שיטה               | תיאור                                                                |
| ------------------ | ------------------------------------------------------------------- |
| **Tailscale Serve** | חשיפת השער דרך רשת Tailscale שלכם (פרטית, מוצפנת)                   |
| **Tailscale Funnel** | חשיפת נתיבים ספציפיים (למשל, `/webhook/*`) לאינטרנט הציבורי         |
| **Cloudflare Tunnel** | ניתוב דרך רשת Cloudflare למופע המקומי שלכם                         |
| **Reverse Proxy**   | הגדרה ידנית עם nginx, Caddy או דומה                                 |

## תצורה (מתוכננת)

```yaml
remote:
  tailscale:
    serve: true
    funnel:
      enabled: true
      paths: ["/webhook/*"]
  auth:
    # אסימון אימות מאוחסן ב-OS keychain
```

## אבטחה

- נקודות קצה webhook הן הנתיבים היחידים שנחשפים כברירת מחדל
- אסימוני אימות מאוחסנים ב-OS keychain
- כל התעבורה מוצפנת במעבר
- ה-WebSocket של השער לעולם אינו נחשף לאינטרנט הציבורי ללא אימות
