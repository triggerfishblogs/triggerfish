# עיון

חלק זה מספק חומר עיון טכני להגדרה והרחבה של Triggerfish.

## [סכמת תצורה](./config-yaml)

עיון מלא ב-`triggerfish.yaml` -- כל חלק, כל מפתח, עם דוגמה מוערת. מכסה
מודלים, ערוצים, סיווג, מדיניות, שרתי MCP, מתזמן והתראות.

## [ממשקים מרכזיים](./interfaces)

עיון בממשקי TypeScript למפתחים הבונים אינטגרציות, מתאמי ערוצים מותאמים,
ספקי LLM או backends אחסון. כולל `StorageProvider`, `ChannelAdapter`,
`LlmProvider`, `NotificationService`, סוגי hooks, מצב סשן ותבנית
`Result<T, E>`.

## [מילון מונחים](./glossary)

הגדרות לכל מונח ייחודי ל-Triggerfish, מ-Buoy עד Write-Down.
