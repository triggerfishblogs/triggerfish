# צעדים ראשונים

Triggerfish היא פלטפורמת סוכני AI מאובטחת ורב-ערוצית עם אכיפת מדיניות
דטרמיניסטית מתחת לשכבת ה-LLM. היא מתחברת לכל פלטפורמת הודעות שאתם כבר
משתמשים בה -- Telegram, Slack, Discord, WhatsApp, דוא"ל, WebChat ו-CLI -- תוך
אכיפת כללי סיווג נתונים שה-AI אינו יכול לעקוף, לדרוס או להשפיע עליהם.

בין אם אתם אדם פרטי שרוצה עוזר AI אישי חזק עם ערבויות אבטחה אמיתיות, ובין
אם ארגון שצריך אכיפת מדיניות ניתנת לביקורת על פני אינטראקציות עובד-סוכן,
Triggerfish נותנת לכם גם את רוחב האינטגרציה וגם את הבקרות להשתמש ב-AI בביטחון.

Triggerfish היא קוד פתוח (רישיון Apache 2.0), רצה באופן מקומי על המכשיר שלכם,
ותומכת במספר ספקי LLM כולל Anthropic Claude, OpenAI, Google Gemini, מודלים
מקומיים דרך Ollama ו-OpenRouter.
