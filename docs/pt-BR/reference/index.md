# Referencia

Esta seccion proporciona material de referencia tecnica para configurar y
extender Triggerfish.

## [Esquema de Configuracion](./config-yaml)

Referencia completa de `triggerfish.yaml` -- cada seccion, cada clave, con un
ejemplo anotado. Cubre modelos, canales, clasificacion, politicas, servidores
MCP, programador y notificaciones.

## [Interfaces Clave](./interfaces)

Referencia de interfaces TypeScript para desarrolladores que construyen
integraciones, adaptadores de canal personalizados, proveedores LLM o backends
de almacenamiento. Incluye `StorageProvider`, `ChannelAdapter`, `LlmProvider`,
`NotificationService`, tipos de hooks, estado de sesion y el patron
`Result<T, E>`.

## [Glosario](./glossary)

Definiciones de cada termino especifico de Triggerfish, desde Buoy hasta
Write-Down.
