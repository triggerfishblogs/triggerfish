# Referencia

Esta sección proporciona material de referencia técnica para configurar y ampliar Triggerfish.

## [Esquema de configuración](./config-yaml)

Referencia completa de `triggerfish.yaml` -- cada sección, cada clave, con un ejemplo anotado. Cubre modelos, canales, clasificación, política, servidores MCP, programador y notificaciones.

## [Interfaces clave](./interfaces)

Referencia de interfaces TypeScript para desarrolladores que construyen integraciones, adaptadores de canal personalizados, proveedores LLM o backends de almacenamiento. Incluye `StorageProvider`, `ChannelAdapter`, `LlmProvider`, `NotificationService`, tipos de hook, estado de sesión y el patrón `Result<T, E>`.

## [Glosario](./glossary)

Definiciones de cada término específico de Triggerfish, desde Buoy hasta Write-Down.
