# Centro de soporte

## ¿Sigue atascado?

Si la documentación anterior no ha resuelto su problema:

1. **Busque issues existentes** en [GitHub Issues](https://github.com/greghavens/triggerfish/issues) para ver si alguien ya lo ha reportado
2. **Pregunte a la comunidad** en [GitHub Discussions](https://github.com/greghavens/triggerfish/discussions)
3. **Abra un nuevo issue** siguiendo la [guía para reportar issues](/es-ES/support/guides/filing-issues)
