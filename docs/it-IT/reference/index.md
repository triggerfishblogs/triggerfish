# Riferimenti

Questa sezione fornisce materiale di riferimento tecnico per la configurazione e
l'estensione di Triggerfish.

## [Schema di Configurazione](./config-yaml)

Riferimento completo per `triggerfish.yaml` -- ogni sezione, ogni chiave, con un
esempio annotato. Copre modelli, canali, classificazione, policy, server MCP,
scheduler e notifiche.

## [Interfacce Principali](./interfaces)

Riferimento delle interfacce TypeScript per sviluppatori che costruiscono
integrazioni, adattatori di canale personalizzati, provider LLM o backend di
storage. Include `StorageProvider`, `ChannelAdapter`, `LlmProvider`,
`NotificationService`, tipi di hook, stato della sessione e il pattern
`Result<T, E>`.

## [Glossario](./glossary)

Definizioni per ogni termine specifico di Triggerfish, da Buoy a Write-Down.
