# Referens

Det här avsnittet tillhandahåller tekniskt referensmaterial för att konfigurera och utöka Triggerfish.

## [Konfigurationsschema](./config-yaml)

Fullständig referens för `triggerfish.yaml` — varje avsnitt, varje nyckel, med ett kommenterat exempel. Täcker modeller, kanaler, klassificering, policy, MCP-servrar, schemaläggare och notifikationer.

## [Nyckelgränssnitt](./interfaces)

TypeScript-gränssnittsreferens för utvecklare som bygger integrationer, anpassade kanaladaptrar, LLM-leverantörer eller lagringsbakändar. Inkluderar `StorageProvider`, `ChannelAdapter`, `LlmProvider`, `NotificationService`, hook-typer, sessionstillstånd och `Result<T, E>`-mönstret.

## [Ordlista](./glossary)

Definitioner för alla Triggerfish-specifika termer, från Buoy till Write-Down.
